Présentation de EAST

EAsy Slide Tool

  • Logiciel de présentaton en Open Source (licence CC BY-NC-SA )
  • Fonctionnalités
    • Création simple de présentation sous forme d'un fichier XML
    • Tranformation du fichier XML en HTML par une feuille XSLT
    • Visualisation du rendu sous un navigateur WEB
1/4

Projet E1 : Diffusion de slides EAST
  • Ce projet est la continuité d'un projet développé les années précedentes
  • Il consiste à permettre la diffusion de présentations dynamiques EAST d'un poste maître à des postes esclaves
  • Il utilise pour cela un serveur Node.js et des Web sockets
  • Travail à réaliser à partir de la version 6 de l'an dernier :
    • Mettre à jour les différents composants : serveur node et modules node
    • Corriger les bugs résiduels
    • Implémenter
      • La possibilité d'utiliser ce service au travers de proxys
      • Une ardoise partagée en SVG
      • Une télécommande sous smartphone
    • Contacts : Didier Courtaud & Philippe Benoit qui a participé au développement de la version 6 l'an dernier
2/4

Problème

    Francais

  • La configuration de EAST n'a pas été trouvée et cela empêche le fonctionnement normal de EAST.
  • Pour corriger ce probleme, placez le répertoire config_EAST à côté de votre fichier HTML dans le même répertoire et rechargez votre fichier HTML

    English

  • EAST configuration has not been found and EAST cannot work normally.
  • To fix this problem, just put the directory config_EAST beside your HTML file in the same directory and reload your HTML file
